Что такое UX/UI дизайн?

UX (User Experience)
UI (User Interface)

UX — это опыт пользователя, который фокусируется на удобстве и удовлетворении от использования продукта.

UI — это пользовательский интерфейс, который отвечает за визуальное оформление и интерактивность.

Основные элементы UX/UI дизайна

Кнопки

Элементы взаимодействия, которые направляют пользователя.

Цветовые схемы

Сочетания цветов, которые создают настроение и улучшают восприятие.

Типографика

Искусство оформления текста для улучшения читаемости.
